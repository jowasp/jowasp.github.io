Projects

OWASP projects is a rich source of materials, which contains free , open source content. OWASP research initiative wants to focus in projects that have a high component of innovation or require further research, with the purpose to find solutions to major problems, security engineers and the security community in general are looking for. The following are some of the projects we are working on. To discuss further about them, please join our group list: https://groups.google.com/a/owasp.org/forum/?hl=en#!forum/owasp-research

We want to conduct an independent research regarding the capabilities of this project. For more information please check : https://www.owasp.org/index.php/Benchmark

OWASP Python Security Project

Python Security is a free, open source, OWASP project that aims at creating a hardened version of python that makes it easier for security professionals and developers to write applications more resilient to attacks and manipulations. The project is designed to explore how web applications can be developed in python by approaching the problem from three different angles We are looking for volunteers to help us work on a harded version. For more information visit: https://www.owasp.org/index.php/OWASP_Python_Security_Project

OWASP ZSC Tool project

OWASP ZSC is an open source software in python language which lets you generate customized shellcodes for listed operation systems. This software could be run on Linux/OSX under python 2.7.x.
Please visit the project page for more info
